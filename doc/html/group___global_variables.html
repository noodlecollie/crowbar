Variable Documentation

CommandLineParser* g_pCmdLine

Global command-line parser object.

The command-line parser is created before the application windows are initialised and holds global application properties.

LogWindow* g_pLog

Global log window object.

Shows debug output from application components. The window still exists if debugging mode is not enabled but does not display, and the Debug menu will not be present in any main application windows.

QList<MainWin*>* g_pWindowTracker

Global window tracker object.

When the last application window closes, the log window should close as well. In order to facilitate this, an application window registers itself to the window tracker on creation and deregisters on close. If the closed window was the last in the list and the log window is still open, it is closed as well.
