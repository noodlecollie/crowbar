#define DECLARE_NUMPROP (   _classname,
  _interface,
  _wrapper,
  _type,
  _proptype,
  _vnull 
)

Declares a number-type property which QString has the capability to convert to and from, in macro form for convenience.

Parameters
_classnameName of the property class.
_interfaceName of the interface it implements.
_wrapperName of the wrapper class to wrap the slots of the interface.
_typeData type returned by getValue (ie. the type the property should hold).
_proptypeThe Property_t value used to classify the property (see baseproperty.h).
_vnullThe default value passed into the property's constructor if no value is specified (eg. 0.0 for float).
#define DECLARE_PROPERTY_INTERFACE (   _classname,
  _wrapper,
  _type 
)

Declares a property interface with an interface classname, wrapper classname and data type (such as int).

Parameters
_classnameName of the interface class.
_wrapperName of the interface wrapper - required to avoid multiple QObject inheritance
_typeVariable type getValue returns.

Parameters

enum Property_t

Property type identifiers.

Used to classify what data type a property holds. When subclassing a new property, define a new identifier here.
