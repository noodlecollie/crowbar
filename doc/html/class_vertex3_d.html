Detailed Description

Defines a vertex in 3D space.

The vertex contains a VBO offset which, when combined with the parent solid's offset, defines where in the VBO this vertex's render data is found. The vertex also contains colour information and texture co-ordinates. Normals are not stored per vertex as when faces are rendered, the face's stored normal is used to calculate the shading required.
