Detailed Description

Class representing a 3D face.

Solids are made up of faces, which are built up from lists of member edges. Faces also contain plane information (for easy conversion to/from VMFs). The direction of the face's edges according to their start/end vertices does not necessarily have to be clockwise (this would be impractical) but the edges do have to form a closed, contiguous loop from the beginning to the end of the list, the last vertex of the last edge being the first vertex of the first edge. The loop should form a convex 2D polygon.
