Detailed Description

Defines a solid class.

Solids are the programmatical implementation of brushes and are made up of sets of vertices, edges and faces. Each edge references the vertices it links, and each face is built from multiple edges. As opposed to Hammer's implementation, solids are mesh-based (ie. they are defined by vertex positions and links rather than planes and intersections) to allow for easier manipulation, and are exported as plane-based objects when serialised as a VMF.
It is recommended to work bottom-up (ie. add any required vertices before edges, and edges before faces) rather than top-down (creating all of a face's edges and vertices as the face itself is created) as the former method is less prone to errors.
